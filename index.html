En este codelab, aprenderás cómo instalar Node.js, configurar Visual Studio Code y crear tu primera aplicación de React utilizando create-react-app.

Paso 1: Instalación de Node.js

  1. Ve al sitio web oficial de Node.js: nodejs.org.
  2. Descarga la última versión LTS (Long-Term Support) de Node.js para tu sistema operativo.
  3. Sigue las instrucciones de instalación en pantalla y acepta los ajustes predeterminados.
  4. Verifica que Node.js se haya instalado correctamente abriendo tu terminal o línea de comandos y ejecutando los siguientes comandos:
    • node -v para verificar la versión de Node.js.
    • npm -v para verificar la versión de npm (Node Package Manager).

Paso 2: Instalación de Visual Studio Code

  1. Ve al sitio web oficial de Visual Studio Code: code.visualstudio.com.
  2. Descarga e instala Visual Studio Code para tu sistema operativo.
  3. Abre Visual Studio Code.

Paso 3: Configuración de Visual Studio Code

  1. Instala las extensiones necesarias para trabajar con React en Visual Studio Code:
    • Abre Visual Studio Code.
    • Ve al menú de extensiones o presiona Ctrl+Shift+X.
    • Busca e instala las siguientes extensiones:
      • ESLint (opcional, pero útil para mantener un código limpio).
      • Prettier - Code formatter (opcional, pero útil para formatear el código automáticamente).
      • Reactjs code snippets (opcional, pero útil para obtener fragmentos de código de React).
  2. Configura Visual Studio Code según tus preferencias personales (tema, atajos de teclado, etc.).

Paso 4: Creación de tu primera aplicación de React con create-react-app

  1. Abre tu terminal o línea de comandos.
  2. Crea una nueva carpeta para tu proyecto de React: mkdir mi-app-react.
  3. Ingresa al directorio de tu proyecto: cd mi-app-react.
  4. Ejecuta el siguiente comando para crear una nueva aplicación de React utilizando create-react-app:
npx create-react-app my-react-app

Reemplaza my-react-app con el nombre que desees para tu aplicación.

Una vez que la instalación haya finalizado, ingresa al directorio de tu aplicación: cd my-react-app. Inicia tu aplicación React:

npm start

Abre tu navegador y ve a http://localhost:3000 para ver tu nueva aplicación de React en funcionamiento.

¡Felicitaciones! Has creado con éxito tu primera aplicación de React utilizando create-react-app y la has configurado para trabajar con Visual Studio Code.
